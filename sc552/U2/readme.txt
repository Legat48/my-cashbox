Для установки агента обновления u2u1 под Windows необходимо:

1) Скопировать на кассу в каталог с файлами UPOS-1 все файлы из каталога U2.

2) Если пинпад подключен через СОМ- или USB-порт, то
  2.1) Номер СОМ-порта, к которому подключен пинпад, указать в файле pinpad.ini как обычно. 
  2.3) Из-под админского аккаунта в этом каталоге дать команду agent.exe /reg
  2.4) Вручную стартовать сервис UPOS 2.0 serial port agent, либо перезагрузить кассу.

3) Убедиться, что адрес 185.157.96.41:80 доступен с кассы.

4) Выполнить сверку итогов для проверки работоспособности. При этом в каталоге должен появиться подкаталог Update. Сверка итогов должна пройти успешно, без ощутимых задержек.
